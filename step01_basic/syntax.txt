1. 자바 확장자 : *.java
2. 실행 process
	개발 -> 컴파일 -> 실행	
3. 컴파일 명령어
	>javac 파일명.java
4. 실행 명령어
	>java 파일명
5. 자바 프로그램 구성 요소
	class
	변수
	생성자
	메소드
	static{}
6. 구성
	package 선언구;
	import 선언구;
	class 프로그램명{
		변수 - 데이터
			타입 변수명;
			타입 변수명 = 값;
			
		생성자 - 메모리에 확보 가능하게 해주는 구성 요소
			클래스명 ([..]){[..]}
			
		메소드 - 데이터 처리하는 기능
			반환타입 메소드명([..]){[..]}
			
			
		static{} - 리소스(메모리..) 절약용 효율적인 구성 요소
	} 
	
7.프로그램 개발시 필요항 구성 요소
	1. 데이터
	2. 데이터 처리하는 기능
	
8. 데이터 타입
	1. 기본
		byte/short/int/long : 0으로 자동 초기화
		float/double : 0.0
		char : '\u0000'
		boolean : false
		- 멤버 변수인 경우 실제 사용 가능한 필수 단계
			: 객체 생성
			: 이 변수들을 보유한 클래스를 타입으로 해서 new 생성자()
		
	2. 참조(객체)
		- 기본을 제외한 모든 타입
		- class 타입 (sun 에서 제시한 클래스/사용자 정의 타입) ex) String/Me
		- 변수가 참조하는 객체가 없을 경우 초기화 되는 기본 값은 ?
			: null
		- 객체 생성 문법
			new 생성자([..])
			단, String과 배열만 new없이도 데이터 표기법만으로 객체 생성
